Shell脚本 备忘

Shell脚本 备忘

  • 解释shell 第一行写#!/bin/sh，在Ubuntu下默认执行的不是bash，而是dash，从下图可以看出，
    /bin/sh是指向dash的。

    而在sh下有的命令是没有的，所以会提示bad substitution。
  • 字符串操作
    • 获取长度：
      echo ${#STRING}
    • 获取子字符串：
      echo ${STRING:m:n} # m，n的头尾都会包含
    • 获取子串索引：
      echo `expr index “$string” g`
  • 流程控制
    • 才知道原来放括号[也是linux里的可执行程序 = =！，所以后头要空格加参数。
      if [ 4 -lt 3 ] then echo 4 else echo 3 fi 参数可以有:eq ge gt le lt ne a o 等
    • 循环 for:
      for var in var1 var2 …
      do
      …
      done
    • while:
      while …
      do
      …
      done
  • 文件包含
    source ./SHELL.sh
    . ./SHELL.sh
    以上两行是等价的，注意第二种写法/不能省略。
  • 获取文件路径
    $ path=`readlink -f shelltest.sh`
    $ echo $path
    /home/chyace/proj/shell/shelltest.sh
