CentOS7在Vmware中应用记录

由于需要搭建基于阿里的Cobar的Mysql集群，所以打算先在虚拟机中实验下效果。

之前因为内存有限，基本上实验是在ArchLinux中的，这次尝试比较大众也比较方便的CentOS。

1.虚拟机迁移问题

昨晚在家里电脑上安装好的CentOS7，选择了带GUI的服务器模式，复制到移动盘里打算在公司用，结果装载不进去，显示：

this kernel requires an x86-64 CPU ,but only detected an i686 CPU. this processor is unsupported in rhel7.

google之发现是BIOS没有开启虚拟化的原因，可以在BIOS设置里选择Intel VirtualTechnology，设置成enable。

2.设置开机默认不开启GUI

打开inittab文件设置

#vim /etc/inittab

在文件末尾加上

id:3:initdefault:

参数3是开机多用户模式，5是图形界面模式。

在CentOS7以前是这种方式，但是7改动比较大，放弃了这种配置方式，inittab文件还在，但是已经不起作用了，提示用新的方法设定：

ln -sf /lib/systemd/system/multi-user.target /etc/systemd/system/default.targtet

通过软链完成设置。

如果想在CLI下打开GUI可以输入：

startx
